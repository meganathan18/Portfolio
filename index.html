Project Highlight

URL Shortener Application

Java | Spring Boot | MySQL

Project Overview

The URL Shortener Application is a high-performance RESTful microservice built to optimize internet address sharing. By converting bulky HTTP links into short 6-letter aliases, it enables cleaner distribution on social feeds, emails, and SMS.

Core Architectural Features

  • Base-62 Collision-Free Encoding: Utilizes a custom conversion scheme to translate incremental database IDs into standard 62-character values (a-z, A-Z, 0-9) to produce compact short codes.
  • Index-Optimized Lookup: Integrates structured index layers in the MySQL database to achieve O(1) time complexity for redirection requests.
  • Analytics Instrumentation: Leverages interceptors to log redirect request headers, capturing details like operating systems, browser agents, geolocation logs, and temporal click counts.
  • Redirection Stats Dashboard: Exposes dedicated API endpoints to compute analytics summaries and render click tracking metrics in real-time.
